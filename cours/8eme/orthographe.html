Leçon 1 : La ponctuation

Corpus

Maman a acheté des légumes, des fruits, du poisson et de la viande.

Où allez-vous?

Quel joli bébé!

Il m'a dit: «Bonjour»

Résumé

Définition

La ponctuation est l'ensemble des signes graphiques qui permettent d'organiser le texte écrit, de marquer les pauses et de préciser le sens des phrases.

Les signes de ponctuation :

Signe Nom Usage
. Le point Termine une phrase déclarative ou impérative
? Le point d'interrogation Termine une phrase interrogative
! Le point d'exclamation Termine une phrase exclamative ou impérative
, La virgule Sépare les éléments d'une énumération, isole un complément
; Le point-virgule Sépare deux propositions liées par le sens
: Les deux-points Introduit une énumération, une explication ou une citation
« » Les guillemets Encadrent une citation ou un dialogue
( ) Les parenthèses Isolent une information complémentaire
... Les points de suspension Marquent une interruption ou une hésitation
- Le tiret Indique un changement d'interlocuteur dans un dialogue

Consolidation

Ponctuez correctement le texte suivant :

Où vas-tu demanda Marie Je vais au marché répondit Pierre Tu peux m'acheter des tomates s'il te plaît

Leçon 2 : L'accord du participe passé employé avec être

Corpus

  1. Les enfants sont partis.
  2. Marie est partie.
  3. Les filles sont parties.
  4. Le livre est tombé.
  5. Les livres sont tombés.

Observation

Avec quel élément le participe passé s'accorde-t-il ?

  • Dans la phrase 1 : partis s'accorde avec les enfants (masculin pluriel)
  • Dans la phrase 2 : partie s'accorde avec Marie (féminin singulier)
  • Dans la phrase 3 : parties s'accorde avec les filles (féminin pluriel)

Le participe passé s'accorde avec le sujet du verbe.

Résumé

Règle

Le participe passé employé avec l'auxiliaire être s'accorde en genre (masculin/féminin) et en nombre (singulier/pluriel) avec le sujet du verbe.

Tableau récapitulatif :

Sujet Accord Exemple
Masculin singulier + ø Il est parti.
Féminin singulier + e Elle est partie.
Masculin pluriel + s Ils sont partis.
Féminin pluriel + es Elles sont parties.

Attention aux verbes pronominaux !

Les verbes pronominaux se conjuguent aussi avec être :

Elle s'est lavée. / Ils se sont rencontrés.

Consolidation

Accordez les participes passés :

  1. Les feuilles sont (tomber) ___________.
  2. Ma sœur est (arriver) ___________ ce matin.
  3. Les garçons sont (partir) ___________ tôt.
  4. Elle s'est (lever) ___________ à sept heures.

Leçon 3 : L'accord du participe passé employé avec avoir

Corpus

  1. J'ai mangé des pommes.
  2. Les pommes que j'ai mangées étaient bonnes.
  3. Elle a chanté.
  4. La chanson qu'elle a chantée était belle.

Observation

Pourquoi le participe passé s'accorde-t-il dans certains cas et pas dans d'autres ?

  • Phrase 1 : COD des pommes est après le verbe → pas d'accord
  • Phrase 2 : COD que (= les pommes) est avant le verbe → accord

Résumé

Règle

Le participe passé employé avec l'auxiliaire avoir s'accorde en genre et en nombre avec le complément d'objet direct (COD) si celui-ci est placé avant le verbe.

Cas possibles :

Position du COD Accord Exemple
COD après le verbe Pas d'accord J'ai mangé des pommes.
COD avant le verbe (pronom relatif) Accord Les pommes que j'ai mangées.
COD avant le verbe (pronom personnel) Accord Je les ai mangées.
Pas de COD Pas d'accord Elle a chanté.

Comment trouver le COD ?

Posez la question : Sujet + verbe + quoi ? / qui ?

J'ai mangé quoi ? → des pommes (COD)

Consolidation

Accordez les participes passés si nécessaire :

  1. Les fleurs qu'il a (cueillir) ___________ sont magnifiques.
  2. Elle a (écrire) ___________ une lettre.
  3. La lettre qu'elle a (écrire) ___________ est longue.
  4. Nous les avons (voir) ___________ hier.

Leçon 4 : L'accentuation

Corpus

  1. élève, étude, médecin, téléphone
  2. père, mère, frère, crème
  3. forêt, fenêtre, bête, tête
  4. où, là, déjà, voilà

Résumé

Définition

Les accents sont des signes placés sur certaines voyelles pour indiquer leur prononciation ou distinguer des homophones.

Les trois types d'accents :

Accent Symbole Usage Exemples
Accent aigu é Se place uniquement sur le e pour produire le son [e] fermé été, école, café, éléphant
Accent grave è, à, ù Sur le e : son [ɛ] ouvert
Sur a et u : distingue des homophones		 père, à (≠ a), où (≠ ou), là (≠ la)
Accent circonflexe â, ê, î, ô, û Indique souvent une lettre disparue (ancien s) ou distingue des homophones forêt, bête, île, hôpital, dû (≠ du)

L'accent grave pour distinguer les homophones

  • à (préposition) ≠ a (verbe avoir)
  • où (lieu) ≠ ou (alternative)
  • là (lieu) ≠ la (article ou pronom)

L'accent circonflexe : trace d'un ancien s

  • forêt ← forest
  • hôpital ← hospital
  • fête ← feste
  • île ← isle
Le tréma (¨)
Le tréma se place sur une voyelle pour indiquer qu'elle se prononce séparément de la voyelle précédente.
Exemples : maïs, Noël, naïf, aïeul

Consolidation

Placez les accents manquants sur les mots suivants :

ecole, pere, foret, cafe, hopital, telephone, deja, ou vas-tu, a bientot
